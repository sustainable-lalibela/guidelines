ArcheoGrid documentation for Sustainable Lalibela project.


## Installation


```Bash
python3 -m venv venv
source venv/bin/activate
pip install mkdocs-material
```

## Start the live-reloading docs server

```Bash 
mkdocs serve
```

## Build the documentation site

```Bash
mkdocs build
```
